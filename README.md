# OllamaDesk
> A local chat interface to interact with language models (LLMs) such as Llama or Mistral via Ollama.

## üß© Project Overview

AI Lab Chat Interface is a web application built with **Vue.js** that allows users to chat with language models running **locally**.  
The goal is to create a simple, extendable interface to experiment with different AI models directly on your computer, without requiring an internet connection.
